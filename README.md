# Sillar Inmobiliaria - Frontend App

Aplicación web moderna y responsiva para la gestión y visualización de propiedades inmobiliarias.

## Tecnologías Principales
- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS** (Estilos y diseño)
- **Axios** (Conexión con API)

## Características
- Catálogo de propiedades interactivo.
- Panel de administración.
- Diseño optimizado para móviles y escritorio.

## Ejecutar localmente

```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev
